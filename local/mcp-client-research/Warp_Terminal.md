# Warp (Terminal)

## Overview
Warp is an AI-powered terminal that supports MCP servers.

---

## Source: Multiple repos

### Configuration
Warp supports MCP servers through its AI features configuration.

### Standard Configuration Format
```json
{
  "mcpServers": {
    "server-name": {
      "command": "npx",
      "args": ["@package/mcp-server"],
      "env": {
        "API_KEY": "your-key"
      }
    }
  }
}
```
