# Windsurf (Codeium) - Accuracy: 4/5

## Accuracy Rating: 4/5
**Consistent across 15+ repos** - Good documentation.

## Summary
Windsurf uses JSON configuration with slightly different key names.

## Verified Configuration

### Config Location
- `~/.codeium/windsurf/mcp_config.json` (or similar path)

### STDIO Format
```json
{
  "mcpServers": {
    "server-name": {
      "command": "npx",
      "args": ["package-name"],
      "env": {
        "API_KEY": "value"
      }
    }
  }
}
```

### HTTP Remote Format (uses serverUrl)
```json
{
  "mcpServers": {
    "server-name": {
      "serverUrl": "https://server.com/sse"
    }
  }
}
```

## Key Difference
- Uses `serverUrl` for HTTP (not `url`)

## Key Notes
- Recommended for remote connections
- Standard mcpServers JSON format
- Restart after config changes
