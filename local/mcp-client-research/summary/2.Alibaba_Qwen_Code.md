# Qwen Code (Alibaba) - Accuracy: 2/5

## Accuracy Rating: 2/5
**Limited documentation** - Similar to Gemini CLI.

## Summary
Qwen Code uses JSON config with httpUrl key.

## Configuration
Config at `.qwen/settings.json`:
```json
{
  "server-name": {
    "httpUrl": "https://server.com/api/mcp"
  }
}
```

## Key Notes
- Uses `httpUrl` (like Gemini CLI)
- Part of Alibaba Qwen ecosystem
- HTTP transport support
