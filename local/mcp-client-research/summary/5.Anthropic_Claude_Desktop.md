# Claude Desktop (Anthropic) - Accuracy: 5/5

## Accuracy Rating: 5/5
**Consistent across 50+ repos** - Most widely documented MCP client.

## Summary
Claude Desktop is the reference MCP client. Configuration is highly consistent across all sources.

## Verified Configuration

### Config File Locations (100% consistent)
- **macOS**: `~/Library/Application Support/Claude/claude_desktop_config.json`
- **Windows**: `%APPDATA%\Claude\claude_desktop_config.json`
- **Linux**: `~/.config/Claude/claude_desktop_config.json`

### Standard Format (100% consistent)
```json
{
  "mcpServers": {
    "server-name": {
      "command": "npx|uvx|python|node|docker",
      "args": ["package-name", "args"],
      "env": {
        "API_KEY": "value"
      }
    }
  }
}
```

### HTTP Remote (via Connectors UI or mcp-remote proxy)
Settings → Connectors → Add Custom Connector
- Name + URL format

Or via mcp-remote:
```json
{
  "mcpServers": {
    "remote": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://server.com/api/mcp"]
    }
  }
}
```

## Key Notes
- Must restart after config changes
- Does NOT inherit shell PATH - use full command paths
- Config must be valid JSON
- Environment variables for sensitive data
