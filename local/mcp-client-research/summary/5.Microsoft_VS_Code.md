# VS Code (Microsoft) - Accuracy: 5/5

## Accuracy Rating: 5/5
**Consistent across 30+ repos** - Official Microsoft support.

## Summary
VS Code has comprehensive MCP support with multiple configuration options.

## Verified Configuration

### Config Locations
- **User Settings**: `settings.json` with `mcp.servers` or `mcp` key
- **Workspace**: `.vscode/mcp.json`

### User Settings Format
```json
{
  "mcp": {
    "servers": {
      "server-name": {
        "type": "stdio",
        "command": "npx",
        "args": ["package-name"],
        "env": {
          "API_KEY": "value"
        }
      }
    }
  }
}
```

### HTTP Remote Format
```json
{
  "mcp": {
    "servers": {
      "server-name": {
        "type": "http",
        "url": "https://server.com/api/mcp"
      }
    }
  }
}
```

### Workspace Format (.vscode/mcp.json)
```json
{
  "servers": {
    "server-name": {
      "command": "npx",
      "args": ["package-name"]
    }
  }
}
```

### Input Prompts for Credentials
```json
{
  "inputs": [
    {
      "type": "promptString",
      "id": "api_key",
      "description": "Enter API Key",
      "password": true
    }
  ],
  "servers": {
    "server": {
      "env": {
        "API_KEY": "${input:api_key}"
      }
    }
  }
}
```

## Key Notes
- Requires `type` field: "stdio" or "http"
- Input prompts for secure credential entry
- Marketplace integration for one-click install
- Commands: `/mcp: List Servers`, `/mcp: Start server`
