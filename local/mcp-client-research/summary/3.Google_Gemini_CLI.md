# Gemini CLI (Google) - Accuracy: 3/5

## Accuracy Rating: 3/5
**Consistent across 5-8 repos** - Uses httpUrl key.

## Summary
Gemini CLI uses JSON configuration with httpUrl for HTTP servers.

## Verified Configuration

### Config Location
- `~/.gemini/settings.json`

### HTTP Format
```json
{
  "mcpServers": {
    "server-name": {
      "httpUrl": "https://server.com/api/mcp"
    }
  }
}
```

Or without wrapper:
```json
{
  "server-name": {
    "httpUrl": "https://server.com/api/mcp"
  }
}
```

## Key Difference
- Uses `httpUrl` (not `url` or `serverUrl`)

## Key Notes
- Part of Google's Gemini ecosystem
- Primarily HTTP transport support
- Config at ~/.gemini/settings.json
