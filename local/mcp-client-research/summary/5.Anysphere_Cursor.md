# Cursor (Anysphere) - Accuracy: 5/5

## Accuracy Rating: 5/5
**Consistent across 40+ repos** - Second most documented MCP client.

## Summary
Cursor has extensive MCP support with both config file and UI options.

## Verified Configuration

### Config File Locations (100% consistent)
- **Global**: `~/.cursor/mcp.json`
- **Project**: `.cursor/mcp.json`

### Standard STDIO Format
```json
{
  "mcpServers": {
    "server-name": {
      "command": "npx|uvx|python",
      "args": ["package-name"],
      "env": {
        "API_KEY": "value"
      }
    }
  }
}
```

### HTTP Remote Format
```json
{
  "mcpServers": {
    "server-name": {
      "url": "https://server.com/sse"
    }
  }
}
```

### Via UI
Cursor Settings → MCP → Add new MCP Server

### Deeplink Installation
Supports one-click installation via deeplinks.

## Key Notes
- Project config overrides global
- Both UI and file-based configuration
- Supports stdio and HTTP transports
