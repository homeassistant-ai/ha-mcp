# GitHub Copilot (GitHub/Microsoft) - Accuracy: 3/5

## Accuracy Rating: 3/5
**Documentation varies by IDE** - Multiple IDE support.

## Summary
GitHub Copilot supports MCP across VS Code, JetBrains, Visual Studio, etc.

## Verified Configuration

### VS Code/Settings
```json
{
  "server-name": {
    "type": "http",
    "url": "https://server.com/api/mcp",
    "tools": ["*"]
  }
}
```

### Config File
`~/.copilot/mcp-config.json`:
```json
{
  "mcpServers": {
    "server-name": {
      "type": "local",
      "command": "npx",
      "tools": ["*"],
      "args": ["@package/mcp-server"]
    }
  }
}
```

### CLI
```bash
/mcp add
```

## Key Difference
- Uses `tools` field to specify available tools
- `type`: "local" or "http"

## Key Notes
- Available across multiple IDEs
- Supports input prompts for credentials
- Tools filtering with ["*"] for all tools
