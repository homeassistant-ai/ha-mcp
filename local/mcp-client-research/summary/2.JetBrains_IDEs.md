# JetBrains IDEs - Accuracy: 2/5

## Accuracy Rating: 2/5
**Limited documentation** - Via AI Assistant plugin.

## Summary
JetBrains IDEs support MCP through AI Assistant plugin.

## Configuration
Via IDE Settings → AI Assistant → MCP configuration

```json
{
  "mcp": {
    "servers": {
      "server-name": {
        "command": "npx",
        "args": ["@package/mcp-server"]
      }
    }
  }
}
```

## Supported IDEs
IntelliJ IDEA, PyCharm, WebStorm, PhpStorm, RubyMine, GoLand, Rider, CLion

## Key Notes
- Requires AI Assistant plugin
- Configuration via IDE settings
- Works with GitHub Copilot integration
