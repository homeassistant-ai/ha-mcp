# Cline (Extension) - Accuracy: 4/5

## Accuracy Rating: 4/5
**Consistent across 10+ repos** - VS Code extension.

## Summary
Cline is a VS Code extension with its own config file location.

## Verified Configuration

### Config Location
- **macOS**: `~/Library/Application Support/Code/User/globalStorage/saoudrizwan.claude-dev/settings/cline_mcp_settings.json`
- **Windows**: `%APPDATA%\Code\User\globalStorage\saoudrizwan.claude-dev\settings\cline_mcp_settings.json`

### STDIO Format
```json
{
  "mcpServers": {
    "server-name": {
      "command": "npx",
      "args": ["package-name"],
      "env": {
        "API_KEY": "value"
      }
    }
  }
}
```

### HTTP Remote Format
```json
{
  "mcpServers": {
    "server-name": {
      "url": "https://server.com/sse",
      "type": "streamableHttp"
    }
  }
}
```

## Key Difference
- Uses `"type": "streamableHttp"` for remote servers

## Key Notes
- Also known as "Claude Dev" extension
- Has built-in UI for managing servers
- Supports both local and remote servers
