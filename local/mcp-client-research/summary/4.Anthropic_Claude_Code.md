# Claude Code (Anthropic) - Accuracy: 4/5

## Accuracy Rating: 4/5
**Consistent across 20+ repos** - CLI-based configuration.

## Summary
Claude Code uses CLI commands for MCP configuration, not config files.

## Verified Configuration

### Basic Commands
```bash
# Add stdio server
claude mcp add <name> <command> [args...]

# Add HTTP server
claude mcp add --transport http <name> <url>

# With environment variables
claude mcp add <name> <command> -e VAR=value

# With scope
claude mcp add <name> <command> --scope user    # Global
claude mcp add <name> <command> --scope local   # Project

# List servers
claude mcp list

# Remove server
claude mcp remove <name>
```

### Examples
```bash
# NPX package
claude mcp add my-server npx @package/mcp-server

# HTTP remote
claude mcp add --transport http api-server https://api.example.com/mcp

# With API key
claude mcp add my-server npx @package/server -e API_KEY=xxx
```

## Key Notes
- CLI-based configuration (no config file editing)
- Supports stdio and HTTP transports
- Scope: user (global) or local (project)
- Can install from MCP marketplace
