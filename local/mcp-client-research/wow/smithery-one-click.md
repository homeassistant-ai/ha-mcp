# Interesting: Smithery One-Click Installation

## Discovery
Smithery (smithery.ai) provides one-click installation for MCP servers across multiple clients.

## How It Works
1. MCP servers register with Smithery
2. Users click installation badge
3. Smithery generates deeplinks/configs for each client
4. Automatic installation for supported clients

## Supported Clients
- Claude Desktop
- Cursor
- VS Code
- Windsurf
- Others

## Relevance for ha-mcp
Consider registering ha-mcp with Smithery for easier installation across all supported clients.
